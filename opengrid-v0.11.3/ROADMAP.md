# Roadmap

## Completed in v0.4 — Plugin foundation

- Plugin registry and administration page
- Heartbeats and metrics
- Reference AIS plugin
- Minimal Artifact proof

## Next

- Plugin configuration forms generated from JSON Schema
- Plugin authentication and scoped credentials
- ADS-B reference plugin
- AIS static-data multipart handling
- Track stale and expiry policy
- Binary Artifact upload
- Artifact relationships in Entity and Task inspectors
- `ExecuteMission` Task referencing a mission Artifact
- Formal plugin SDK helpers and conformance tests


## After v0.7

- Command acknowledgement and vehicle-specific completion verification
- Arm, disarm, takeoff and land tasks with explicit safety gates
- MAVLink mission Artifact upload and ExecuteMission
- Parameter inspection and controlled parameter updates
- Vehicle connection routing examples using mavlink-router
- Open icon-pack registry with license metadata
