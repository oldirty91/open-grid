# Public Lattice Concept Mapping

OpenGrid intentionally follows the public conceptual simplicity of Lattice without attempting to reproduce private implementation details.

| Public Lattice concept | OpenGrid |
|---|---|
| Entity | Entity |
| Asset | Entity with `ontology.template = ASSET` |
| Track | Entity with `ontology.template = TRACK` |
| Geo Entity | Entity with `ontology.template = GEO` |
| Components | Entity components |
| Task | Task |
| Taskable agent | Entity that advertises a Task Catalog |
| Task Catalog | `task_catalog` entity component |
| Object | Artifact |
| Object service | Artifact service, planned for v0.3 |

OpenGrid uses its own schemas, implementation and user experience.
